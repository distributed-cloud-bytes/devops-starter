# Security policy

## Supported versions

| Version | Supported |
|---------|-----------|
| `main` | Yes |

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

1. Open a private [GitHub security advisory](https://github.com/distributed-cloud-bytes/devops-starter/security/advisories/new) on this repository, or
2. Contact the maintainers via your organization's preferred channel.

We will acknowledge reports within a reasonable timeframe and coordinate a fix before public disclosure when appropriate.

## Scope

This repository ships local development Compose and configuration templates. Production hardening (TLS, Kafka ACLs, secrets management) is the responsibility of the deploying team.
